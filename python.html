Filter and Reduce functions

filter function -->
    l=[1,2,3,4,5,6]
    def greater(num):
     return num>5
a= list(filter(greater , l))
print(a)

it will return 6

Reduce function -->
    from functools import reduce
    list = [1,,3,4,5,6]
    num = reduce(lambda x,y : x+y , list)
    print(num)
